# Vanjs API文档

## 实例方法

### $destroy
销毁组件

### $addEventListener
添加鼠标事件监听器

### $emit
触发冒泡消息事件

### $dispatch
触发派发消息事件

### $registHandler
注册事件处理器

### $unregistHandler
取消已注册的事件

### $broadcast
触发广播事件

### $mount
挂载子组件

### $unmount
卸载子组件

### Component.prototype.newInstance
创建组件实例

### Component.prototype.newOffInstance
创建组件实例

### Component.prototype.extends
继承组件
